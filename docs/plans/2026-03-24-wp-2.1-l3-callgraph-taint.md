# WP 2.1: L3 Call-Graph Taint ‚Äî Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add intra-module call-graph taint propagation (Level 3) ‚Äî refining function-level taints by analyzing what functions call.

**Architecture:** Three-way provenance classification (decorator/module_default/fallback), total trust order via `least_trusted()`, SCC-based fixed-point propagation, mandatory taint provenance tracking.

**Tech Stack:** Python 3.12+, stdlib `ast` only (no `astroid`), Hypothesis for property-based tests

**Prerequisites:**
- v0.2.1 shipped (corpus verifier taint wiring, analysis_level_required)
- Design spec approved: `docs/superpowers/specs/2026-03-24-l3-callgraph-taint-design.md`

**Key design decisions (from two rounds of 7-reviewer panel):**
- Decorated functions are **anchored** (immutable seeds) ‚Äî core security property
- Module_tiers functions are **floating-downward** ‚Äî can be demoted but not upgraded
- Fallback (UNKNOWN_RAW) functions are **floating-unconstrained** ‚Äî L3's primary value
- Unresolved calls are **transparent** (no taint contribution) with `L3-LOW-RESOLUTION` diagnostic
- Provenance is **mandatory** (assessment evidence, not optional)

## Task 2: Trust Order + `least_trusted()`

**Files:**
- Create: `src/wardline/scanner/taint/callgraph.py`
- Create: `tests/unit/scanner/test_callgraph.py`

**What:** Define the total trust order and the `least_trusted(a, b)` function.

**Implementation:**
```python
TRUST_RANK: dict[TaintState, int] = {
    TaintState.AUDIT_TRAIL: 0,
    TaintState.PIPELINE: 1,
    TaintState.SHAPE_VALIDATED: 2,
    TaintState.UNKNOWN_SEM_VALIDATED: 3,
    TaintState.UNKNOWN_SHAPE_VALIDATED: 4,
    TaintState.EXTERNAL_RAW: 5,
    TaintState.UNKNOWN_RAW: 6,
    TaintState.MIXED_RAW: 7,
}

assert len(TRUST_RANK) == len(TaintState), "TRUST_RANK must cover all TaintState members"

def least_trusted(a: TaintState, b: TaintState) -> TaintState:
    """Return the less-trusted of two taint states (higher rank)."""
    return a if TRUST_RANK[a] >= TRUST_RANK[b] else b
```

**Tests:**
- `test_trust_rank_covers_all_states` ‚Äî every TaintState has a rank
- `test_least_trusted_returns_less_trusted` ‚Äî `least_trusted(AUDIT_TRAIL, EXTERNAL_RAW)` ‚Üí `EXTERNAL_RAW`
- `test_least_trusted_symmetric` ‚Äî `least_trusted(a, b) == least_trusted(b, a)` for all pairs
- `test_least_trusted_identity` ‚Äî `least_trusted(a, a) == a` for all states
- `test_mixed_raw_is_bottom` ‚Äî `least_trusted(MIXED_RAW, X) == MIXED_RAW` for all X

**Definition of Done:**
- [ ] `TRUST_RANK` assertion passes at import time

## Task 3: Call Graph Extraction

**Files:**
- Modify: `src/wardline/scanner/taint/callgraph.py` (add to file from Task 2)
- Modify: `tests/unit/scanner/test_callgraph.py` (add to file from Task 2)

**What:** `extract_call_edges(tree, qualname_map) -> tuple[dict[str, set[str]], dict[str, int], dict[str, int]]` ‚Äî returns adjacency list, resolved call counts per function, and unresolved call counts per function.

**Qualname map:** Reuse `ScanEngine._build_qualname_map()`. Extract to `scanner/_qualnames.py` if not already shared (check if Task already done ‚Äî the shared `make_rules()` extraction may have created this). The function returns `dict[int, str]` keyed by `id(node)`.

**Resolution rules:**
1. Build reverse maps: `{bare_name: qualname}` for module-level defs, `{class_name: {method_name: qualname}}` for methods
2. For each function body, walk `ast.Call` nodes:
   - `ast.Name(id=X)` where X in module defs ‚Üí resolved edge
   - `ast.Attribute(value=ast.Name(id='self'), attr=M)` ‚Üí look up class from caller qualname prefix ‚Üí resolved edge if found
   - `ast.Name(id=ClassName)` where ClassName is a class ‚Üí edge to `ClassName.__init__` if exists
   - Everything else ‚Üí unresolved (increment count, no edge)

**Use iterative AST walk** (not recursive) to avoid recursion limit on large function bodies. Use `ast.walk()` which is iterative.

**Tests:**
- `test_direct_call_resolved` ‚Äî `foo()` where `def foo` exists ‚Üí edge
- `test_self_method_resolved` ‚Äî `self.process()` in class method ‚Üí edge to `ClassName.process`
- `test_constructor_resolved` ‚Äî `MyClass()` ‚Üí edge to `MyClass.__init__`
- `test_import_call_unresolved` ‚Äî `json.loads()` ‚Üí no edge, unresolved count incremented
- `test_parameter_call_unresolved` ‚Äî `callback()` where callback is a parameter ‚Üí no edge
- `test_nested_function_call` ‚Äî inner function calling outer function ‚Üí resolved
- `test_no_duplicate_edges` ‚Äî same call twice ‚Üí single edge (set semantics)
- `test_resolved_unresolved_counts` ‚Äî correct counts returned

**Definition of Done:**
- [ ] All resolution patterns tested
- [ ] Resolved/unresolved counts accurate
- [ ] No recursive AST walk (iterative only)
